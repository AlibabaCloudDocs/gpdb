# ST\_attrFloatMin

获得轨迹属性字段类型为float的属性最小值。

## 语法

```
float8    ST_attrFloatMax(trajectory traj, cstring attr_field_name);
```

## 参数

|参数名称|描述|
|----|--|
|traj|需要获得属性信息的轨迹。|
|attr\_field\_name|指定的属性字段名称。|
