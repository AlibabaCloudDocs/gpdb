# ST\_DateTime

获得栅格对象或波段的时间。

## 语法

```
text ST_DateTime(raster raster_obj);
timetamp ST_DateTime(raster raster_obj,integer band);
```

## 参数

|参数名称|描述|
|:---|:-|
|raster\_obj|需要计算的raster对象。|
|band|波段序号，取值从0开始。|

## 描述

获得栅格对象或波段的时间。如果指定波段，则返回对应波段的时间信息；未指定波段，则返回包含所有波段时间信息的JSON字符串。
